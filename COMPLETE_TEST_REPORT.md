## ⚠️ 已知问题

### 1. FTS5 集成测试依赖问题

**问题描述**: 9 个 FTS5 测试失败，错误信息 `no such table: main.mft`

**根本原因**: FTS5Search 类初始化时依赖 MFT 表，但测试使用独立内存数据库

**影响范围**: 仅影响单元测试，不影响实际使用（生产环境使用统一数据库）

**解决方案**: 
- 短期：跳过 FTS5 独立测试，使用集成测试验证
- 长期：重构 FTS5Search 使其可独立初始化

**优先级**: 低（不影响核心功能）

---

### 2. AssemblerV2 导入问题

**问题描述**: `ImportError: cannot import name 'Slice' from 'mfs.assembler_v2'`

**影响文件**: test_assembler_v2.py, test_stress.py

**解决方案**: 更新测试文件以匹配当前 API

**优先级**: 中（影响压力测试运行）
