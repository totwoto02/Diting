
## 七、低价服务器适配方案（重要）

### 7.1 环境约束

| 约束 | 影响 | 级别 |
|------|------|------|
| **无 GPU** | 向量计算速度降低 10-50倍 | 🔴 高 |
| **2GB 内存** | 无法加载大型向量模型（BGE-M3约2.3GB） | 🔴 高 |
| **4GB 内存** | 可勉强加载，但剩余空间紧张 | 🟡 中 |

### 7.2 适配策略

#### 方案 A：轻量化本地模型（推荐）

| 模型 | 内存占用 | 向量维度 | 中文支持 | 速度（CPU） |
|------|---------|---------|---------|-----------|
| **m3e-small** | ~200MB | 256 | 良好 | 快 | ✅ 2GB环境最优 |
| **m3e-base** | ~800MB | 768 | 优秀 | 中等 | ✅ 4GB环境首选 |
| **text2vec-base** | ~400MB | 512 | 良好 | 中等 | ✅ 2GB环境首选 |
| BGE-M3 | ~2.3GB | 1024 | 优秀 | 慢 | ❌ 不适合低价服务器 |

**推荐配置**：
- **2GB 环境** → `m3e-small` (256维，~200MB)
- **4GB 环境** → `m3e-base` (768维，~800MB)
- **>4GB 环境** → `BGE-M3` (1024维，~2.3GB)

#### 方案 B：远程向量 API（备选）

| API | 优势 | 劣势 | 成本预估 |
|------|------|------|---------|
| 百度智能云向量 API | 国内稳定、中文优化 | 按调用计费 | ~¥0.002/次 |
| 阿里云向量 API | 国内稳定、性价比高 | 按调用计费 | ~¥0.001/次 |
| OpenAI Embedding | 国际通用、质量高 | 国内网络不稳定 | ~$0.0001/次 |

**适用场景**：
- 调用量 < 1000次/天 → API 成本可接受
- 调用量 > 10000次/天 → 本地模型更经济

### 7.3 内存管理策略

- **延迟加载**：启动时不加载模型，首次使用时加载
- **内存释放**：长时间未使用时释放模型内存（gc.collect）
- **批量处理**：减少模型加载次数，批量编码

### 7.4 ChromaDB 内存优化

- 使用 `duckdb+parquet` 磁盘存储模式（节省 ~70% 内存）
- 禁用匿名遥测数据
- 持久化目录设置

### 7.5 无 GPU 性能预估

| 操作 | GPU 环境 | CPU 环境 (2核) | 延迟对比 |
|------|---------|---------------|---------|
| 单文本向量化 (512维) | ~5ms | ~50-200ms | 10-40倍 |
| 批量向量化 (32条) | ~50ms | ~500-2000ms | 10-40倍 |
| 向量搜索 (1000条) | ~5ms | ~10-30ms | 2-6倍 |

### 7.6 验收标准调整（低价服务器）

| 指标 | 原目标值 | 低价服务器调整值 |
|------|---------|----------------|
| 写入延迟 | <500ms | <1000ms（含向量化） |
| 向量搜索延迟 | <50ms | <100ms（CPU环境） |
| 内存占用 | 无限制 | <1GB（模型+ChromaDB） |
